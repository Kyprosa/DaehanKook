Moon Reader+,
Material Concept Re-Design

Fixing the outmoded features of old apps

As a text reader, Moon Leader+, an application that has long been a high market share in the Android market, has been outdated in design and has many inconveniences, such as moving to 2 Debs to set options. I tried to carry out the latest design by utilizing the latest design trend, the Merial Design Concept.

Basically, contents that took up only the online library, donation, advertisement, etc., which were in the user evaluation contents, were boldly removed from the first screen or placed at the bottom of the side menu, and a straight line interface related to path memory, and visible parts were to be solved in order to solve the difficulties of the internal memory path search process.

Finally, through the Play Store account linkage, we added an account drive function to enable multi-device connections such as smartphones, e-books, and tablets.
